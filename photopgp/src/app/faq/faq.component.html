Frequently Asked Questions

Installation and Setup

How to install and setup

  • Before using PhotoPGP you will first need to install OpenKeychain. OpenKeychain is an implementation of OpenPGP for Android (see LINKS) - available from the Google Play Store, F-Droid, or GitHub (as open-source). PhotoPGP uses the openpgp-api service of OpenKeychain for secure key handling and cyptographic support.
  • You will also need to create your own PGP private key (if you don't already have one) for encryption / decryption and signature / verification. There are many ways to create your PGP private key. You can, for example, create your PGP key using Windows or Linux, then copy that key to your phone. Or you can create your key using the OpenKeychain app, then backup your key off your phone.

Import Keys into OpenKeychain

  • Two keys are available in the Public PhotoPGP Directory: (1) PhotoPGP Public Key, (2) Demo Private Key (password: "demo").
  • In the INDEX Tab, click on the KEY Icon to import each key into OpenKeychain.
  • If you have the PhotoPGP Full Edition, you will want to download and import more public-keys into OpenKeychain for anyone you want to authorize to decrypt specific photos.
  • Try the Ubuntu Public Keyserver: https://keyserver.ubuntu.com/

How to take pictures

How to take pictures - Method 1 - Use the INDEX tab, click "ADD PHOTO" icon

  • Review what keys you have selected in the SETTINGS tab, because one of these keys will be required to view the photograph.
  • Select the directory/folder where you want the encrypted images (set directory, new folder, click folder icon)
  • Click the INDEX tab ADD PHOTO icon to start the standard camera interface - click the shutter button, then click check.
  • The thumbnail image will appear, click on the thumbnail image (to decrypt) or select VIEWER tab.
  • If the image is rotated, click the rotation icon to change - this also changes the thumbnail
  • Method 1 - this uses the standard camera app with a secure file provider interface to produce an encrypted jpg file - without compression.

How to take pictures - Method 2 - Use the CAMERA tab

  • Review what keys you have selected in the SETTINGS tab, because one of these keys will be required to view the photograph.
  • Select the directory/folder where you want the encrypted images (set directory, new folder, click folder icon)
  • Click CAMERA tab, select which camera, and other options.
  • This method automatically rotates the image, provides options to control the picture quality (compression), and captures additional meta data (orientation, keys, etc.)
  • Method 2 - this uses a low level camera api with controls on image resolution to produce (multiple) encrypted compressed jpeg or png files, and meta files.
