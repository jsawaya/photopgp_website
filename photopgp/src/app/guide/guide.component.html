PhotoPGP Quickstart:

SETTINGS TAB

First-Time Setup

  • Allow Device Permissions (Camera and Write_External_Storage).
  • OpenKeychain Installation and Import your private key

OpenPGP Features

  • Select your private key (from those you imported into OpenKeychain).
  • Optional, you can select public keys to authorize recipients to decrypt files.
  • Select your preferred encryption file format - ASCII Armor or PGP binary.

More Features

  • Enable Toast Messages. Enable Vibration. Vintage Device (Lower Resolution).
  • Google SEND action to Gmail or Drive.

INDEX TAB

Click thumbnail/icon to view that selected item.

Click description to open the ACTION screen.

Select Directory

  • Select Public or Private Directory.

New Folder

  • Creates a new sub-folder in the current directory/folder.

New Drawing

  • Creates a new encrypted drawing file

New Note

  • Creates a new encrypted text file (which may be viewed/edited by selecting the comment icon)

New Photo

  • Creates a new encrypted image file (which may be viewed by selecting image icon)
  • This camera interface is different from the CAMERA tab:
  • Provides standard camera features (zoom, flash), automatic encryption or secure private storage
  • Creates a new encrypted text file (which may be viewed/edited by selecting the comment icon)
  • The CAMERA tab has other features (auto-rotation, picture quality, and orientation data)

Up / Back Button

  • Allows you to traverse back up the directory tree to the parent directory

Supports the following file types:

  • images (jpg/jpeg/png)
  • encrypted images (asc/pgp)
  • notes - encrypted text
  • sub-folders
  • ZIP files
  • keys

ACTION SCREEN

TRASH

  • delete image - also deletes an associated note, signature, rotation and meta files
  • delete folder - also deletes any associated note, with option to delete folder contents

MOVE

  • Move the item to another directory/folder
  • Across public/private directories - the item will be copied (not moved)

NAME

  • Renames the item to a new unique id [date_time_random]

CLOSE

  • Close the ACTION screen and return the INDEX (or use back button)

NOTE

  • Add/edit encrypted text note to the item
  • You can also sign it, if this is an existing note.
  • If there is a signature, you can verify it and see what key was used to sign it

SEND

  • Send the item to GMAIL or Google Drive...

ZIP / UNZIP

  • If the selected item is a folder, you can copy it to backup ZIP file
  • If the selected item is a ZIP file, you can restore the archived contents back to a folder

DECRYPT / ENCRYPT

  • If the item is an encrypted image (jpg/jpeg/png) you can decrypt it
  • If the item is an image (jpg/jpeg/png), you can encrypt it
  • IMPORTANT: You don't need to decrypt images as the VIEWER tab automatically does this in memory. The only good use case for this is to decrypt an image into the Private PhotoPGP directory, in order to re-encrypt it with the public keys for the recipients of the encrypted image.

VIEWER TAB

Double-click, pinch and drag gestures to zoom/pan image.

Trashcan Icon

  • Deletes this image (then shows the next image).

Drawing Icon

  • Add drawing annotion to image.
  • Long-click crops the image and re-encrypts with current key settings.

Rotation Icon

  • Rotates the image left or right, also rotates the thumbnail image

Security Check Icon

  • Adds a detached signature, or verifies a previous signature

Play Action Icon

  • Opens the ACTION screen for this item.

CAMERA TAB

Click the preview area to capture an encrypted image, or long-click the area below the preview.

FORWARD / REVERSE / OFF

  • Select which camera to use.

HD Icon

  • Select which resolution

Aspect Ratio

  • Select which ratio makes the preview looks best on your device.

Picture Quality / Compression

  • Select your preferred picture quality (image size)

Preview Size

  • Select the preview window size that looks best.

Burst Count

  • Select how many photos you want to take each time you click the shutter button.

Focus

  • Select focus to update the preview window.

Shutter Button

  • Select the shutter button, or anywhere on the image, to take photo.
